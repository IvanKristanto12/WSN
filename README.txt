Cara penggunaan Program:
1. Setup sensor node sebagai base station di file BSManager.java bagian attribut "connectedNodeAddr".
2. Setup sensor node sebagai node di file SNManager.java bagian attribut "sensorId", "SENSOR_NODE_ADDRESS", "baseStationAddr", "nextNode", dan "PreviousNode"
2. Jalankan Tester.jar
3. Check Online dengan input '1'
4. Jika semua sensor sudah "ONLINE", masukan input '2'.
   Akan muncul Folder "SenseResult" di lokasi Tester.jar sebagai tempat menyimpan hasil
5. Untuk memberhentikan sensing, masukan input '3'
6. Untuk memberhentikan program, masukan input '4'
